# 圣杯/永久投资组合

我在过去几年, 投资方式已经从量化交易转向了更长期的持有. 促使这一转变的原因有很多, 其中包括对市场波动性的厌恶, 以及对长期财富积累的兴趣.

简单的说, 现在我已经不会再幻想一夜翻倍, 或者一年赚多少这些不切实际的目标了. 相反, 我更关注于通过稳健的投资策略来实现财富的**长期增长**. 投资三要点:

0. 不要亏
0. 不要亏
0. 不要亏

我现在的策略基础是哈利·布朗在 20 世纪 80 年代提出的永久投资组合. 该策略旨在通过将资金平均分配于股票, 长期国债, 黄金和现金(或短期国债)四种资产, 实现长期, 稳定且低回撤的投资回报, 以应对繁荣, 衰退, 通胀和通缩这四种不同的经济环境. 该组合的特点是简单, 安全, 稳定, 投资者无需预测经济走势或进行择时操作, 只需按照既定比例持有资产.

由于永久投资组合提出的年代较早, 当时并无加密货币这个资产类别. 因此我在实践这个策略时, 将加密货币作为第五种资产类别纳入组合, 并在内部对其进行了细分, 包括比特币, 以太坊和其他山寨币. 因此我实际的资产配置比例为:

- 传统资产共占比 1/2.
    - 纳斯达克和标准普尔指数, 占比 1/3.
    - 美元现金理财, 占比 1/3.
    - 黄金, 占比 1/3.
- 加密货币共占比 1/2.
    - 比特币, 以太坊等主流, 占比 1/2.
    - 美元稳定币的现货理财, 占比 1/2.

我将美元现金理财和美元稳定币理财视为哈利·布朗所认为的长期国债和现金, 因为它们都属于低风险, 高流动性资产, 本质上没有太多区别. 因此我的组合里现金类资产占比为 1/6 + 1/4 = 41%. 该比例略低于哈利·布朗建议的 25% + 25%. 但考虑到我个人的风险偏好和对加密货币的信心, 我认为这是合理的调整.

最后谈一下我对未来的看法. 我认为未来的经济环境将更加复杂和多变, 传统资产会面临加密货币资产的强烈冲击. **尤其是对于现在的第三世界国家, 其本币的贬值和通胀问题将更加严重, 加密货币资产在这些国家的平民眼里吸引力会更大, 因此去投资这些国家的传统资产类别可能没有太多意义.** 近一年国际上已经发生过多次大规模平民抗议事件, 当政府进行强力镇压, 社会秩序失控时, 这些国家的平民会更倾向于持有加密货币而非本币. 我曾经长时间投资越南的股市, 但现在我已经全部卖出, 如果因为美国扶持而投资一个国家, 为什么不直接投资美国呢? 这才是更简单, 更有效的方式.
