# 带阻和带通滤波

带阻滤波器(BR)传递函数:

\ | 理想 | 巴特沃斯 | 高斯
- | --- | --- | ---
H | $$H(u, v) = {(0, D_0 - frac{W}{2} <= D <= D_0 + frac{W}{2}),(1, text(else)):}$$  | $$H(u, v) = frac{1}{1 + [frac{DW}{D^2 - D_0^2}]^{2n}}$$ | $$H(u, v) = 1 - e^-[frac{D^2 - D_0^2}{DW}]^2$$

其中 $$W$$ 是带宽, $$D$$ 是 $$D(u, v)$$ 距离滤波中心的距离, $$D_0$$ 是截止频率, $$n$$ 是巴特沃斯滤波器的阶数.

一个带通滤波器(BP) 的传递函数是

$$
H_{BP}(u, v) = 1 - H_{BR}(u, v)
$$
