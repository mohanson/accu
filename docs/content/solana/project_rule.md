# Solana/在主网发行您的代币/设计空投规则

在主网运营一个代币项目, 代币分发一直是个大难题. 空投是一种流行的大幅度增加代币持有人数的方式, 可以帮助新项目吸引用户.

我为 pxs 代币设计了一种空投方式. 不同于传统的空投方式, pxs 代币的空投被计划空投给"学习过此教程"的用户. 空投通过智能合约自动分发的, 没有手动转账的繁琐过程. 用户只需支付一笔非常低的 solana 网络手续费, 即可免费领取固定数量的 pxs 代币.

Pxs 的空投遵循两个设计原则: 一是合约自动空投, 二是无需许可.

一些传统项目的空投, 通常需要项目方手动将代币发送到用户钱包, 但 pxs 代币的空投应该完全通过合约程序实现. 空投过程完全去中心化, 每个符合条件的用户都可以通过合约领取代币, 而不需要依赖任何中介.

同时与许多空投项目要求用户完成任务, 提交 kyc 等不同, pxs 空投没有任何权限限制, 任何人都可以无限制的领取. 您只需要连接到 solana 网络, 支付一定的手续费, 就能领取代币.

不过, 请注意, 理论上, 任何人都可以领取任意数量的 pxs 代币, 但实际限制是 solana 网络的手续费. 换句话说, pxs 的实际价格等于一个 solana 交易的手续费.
